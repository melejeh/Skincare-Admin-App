# PlanetMel — Admin Desktop App

A JavaFX + Spring Boot desktop application for managing the PlanetMel Skin Co. product catalogue and orders.

## Features
- Secure login with role-based access
- Add, edit, and delete products
- View and manage customer orders
- Shared H2/PostgreSQL database with the web app

## Tech Stack
- Java 17
- JavaFX
- Spring Boot
- Spring Data JPA
- H2 Database

## Part of the PlanetMel Platform
- [SkinCare-Website](https://github.com/melejeh/SkinCare-Website) — static frontend
- [SkinCare-WebApp](https://github.com/melejeh/SkinCare-WebApp) — customer-facing web app
